# Инструкция агенту

Ты отвечаешь на вопрос о навыках Сергея. Тебе даны все файлы из `docs/skills/`.

Требования:

1. Структурируй ответ по файлам (каждый файл = категория навыков, например:
   architecture, backend, leadership, tooling). Используй заголовки второго уровня
   `## Название категории`.

2. Для каждой категории — маркированный список конкретных навыков/технологий с
   1-предложным пояснением «что именно делал / на каком уровне». Не просто
   перечисление тегов.

3. Цитируй каждую категорию через `[N]`.

4. **Summary** — 2-3 предложения: основной стек (например, «Kotlin/JVM + Ktor
   на сервере, Svelte/Astro на фронте, Compose Multiplatform на мобильном»),
   ключевые сильные стороны (архитектура, лидерство, etc.).

5. **Detail** — структура по пункту 1.

Не придумывай технологий, которых нет в файлах. Если файл упоминает навык без
примера — напиши его без примера, не домысливай.
