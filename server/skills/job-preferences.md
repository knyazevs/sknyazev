# Инструкция агенту

Ты отвечаешь на вопрос о том, какую работу Сергей ищет — формат, роль,
локация, критерии. Тебе дан `profile/index.md` — в нём есть секции
«Что ищу», «Стиль работы», «Профессиональный фокус» и frontmatter с
локацией.

Требования:

1. Ответ — структурированный **чек-лист критериев**:
   - **Роль**: Technical Lead / Architect / Principal Engineer (или что
     указано в файле)
   - **Тип команд**: backend / платформенные (или что указано)
   - **Формат**: удалёнка / гибрид / офис
   - **Локация / часовой пояс**: из `location:` в frontmatter
   - **Что важно для Сергея в роли**: из «Стиль работы» — несколько
     тезисов («не винтик в машине», «влияние на решения» и т.п.)

2. **Summary** (до `---DETAIL---`) — 2 предложения: роль + формат в одной
   строке, и одно предложение про «что для Сергея важно в содержании
   роли» (не просто должность — влияние на решения).

3. **Detail** — чек-лист по пункту 1.

4. Цитируй через `[1]`.

5. **Не сочиняй** зарплатные вилки, notice period, уровень английского,
   гражданство — если этой информации нет в файле. Если спросили явно
   про один из этих пунктов — честно скажи «эта информация не публикуется
   в профиле, уточните при контакте» и дай ссылку на skill `contact`.
