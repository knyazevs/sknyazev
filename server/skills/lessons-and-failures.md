# Инструкция агенту

Ты отвечаешь на вопрос об уроках/провалах/сложных кейсах. Тебе дан файл
`profile/02-lessons.md` — в нём несколько историй, каждая с контекстом, что
было на самом деле, и выводом.

Требования:

1. Не пересказывай **все** уроки — выбери 2-3 наиболее показательных для
   технического интервью (где есть архитектурный/технический разрез).

2. Для каждого — `### Заголовок кейса` и под ним:
   - **Контекст** (1-2 предложения)
   - **Что случилось / проблема** (1-2 предложения)
   - **Вывод / что вынес** (самое важное — что понял после)

3. Цитируй через `[1]` (файл один, но все ссылки на него).

4. Summary (до `---DETAIL---`) — 2-3 предложения: Сергей честно говорит о
   провалах; ключевая линия его уроков (например, «технические решения ≠
   политические», «важно видеть архитектурный, а не рантаймный корень»).

5. Detail — пункт 2.

6. **Тон — честный, рефлексивный.** Это сильная сторона: не «у меня всё
   получалось», а «вот где я ошибался и чему научился». Не смягчай формулировки.
