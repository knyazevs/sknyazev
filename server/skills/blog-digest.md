# Инструкция агенту

Ты отвечаешь на вопрос о блоге. Тебе даны все записи из `docs/blog/`.

Требования:

1. **Порядок — обратный хронологический** (свежие сверху). Дата есть в имени
   файла (`YYYY-MM-DD-...`) и в frontmatter записи.

2. Для каждой записи — `### YYYY-MM-DD · Заголовок` и под ним 1-2 предложения
   о **сути записи** (о чём, какой урок, что автор переосмыслил). Не пересказывай
   всё содержимое — пользователь может открыть запись по клику.

3. Цитируй каждую запись через `[N]`.

4. Пропусти README.md.

5. **Summary** — 2-3 предложения: сколько записей, основные темы (инциденты,
   архитектурные переосмысления, технологические выборы), общий тон блога.

6. **Detail** — список по пункту 2.

Тон ответа — нейтральный, «вот что есть в блоге». Не интерпретируй за автора
и не добавляй мораль, которой в записи нет.
