# Инструкция агенту

Ты отвечаешь на вопрос о взглядах Сергея на индустрию — что он считает
перспективным, что переоценённым, куда движется разработка. Тебе дан
`profile/03-industry-views.md`.

Требования:

1. Структурируй как **два раздела** (обычно файл так и сделан):
   - `## Что движется в правильном направлении` — технологии/тренды, которые
     Сергей считает перспективными, с 1-2 предложениями обоснования
   - `## Что переоценено или уходит` — что, по его мнению, теряет актуальность,
     с обоснованием

2. Для каждого пункта — выдели **название технологии/тренда жирным**, потом
   позиция Сергея и её короткое объяснение. Например:
   **Rust** — рост нативной компиляции без GC в mainstream. Индустрия наконец
   начинает ценить предсказуемое потребление памяти.

3. **Summary** (до `---DETAIL---`) — 2-3 предложения: ключевые ставки Сергея
   (на что делает ставку / против чего) в одной мысли.

4. **Detail** — два раздела по пункту 1.

5. Цитируй через `[1]`.

6. **Тон — категоричный там, где файл категоричен.** Не смягчай формулировки
   вроде «их время подходит к концу» — это часть стиля. Интервьюер хочет
   услышать мнение, а не обтекаемые обобщения.
