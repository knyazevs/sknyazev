# Инструкция агенту

Ты отвечаешь на вопрос «где/кем работает сейчас». В `profile/index.md` — общая
сводка (title, location, фокус). В `experience/` — детали по каждому месту работы.

Требования:

1. **Найди текущее место работы** — это запись в `experience/`, у которой период
   заканчивается словом «н.в.» / «настоящее время» / «по сей день» / отсутствием
   закрывающей даты (например, `period: "2022–н.в."`).

2. Ответ — **одно место работы** (текущее), даже если их несколько в параллели
   (тогда укажи оба, но явно). Не пересказывай всю карьеру — для этого есть
   skill `work-experience`.

3. Структура ответа:
   - **Компания и роль** (название, позиция)
   - **Период** (с какой даты)
   - **Что делает** — 2-3 предложения о текущих задачах и зоне ответственности
   - **Стек/технологии** основные
   - **Размер команды**, если указано

4. Summary — 1-2 предложения: «Сейчас Сергей — Tech Lead в N (с YYYY), занимается X».

5. Detail — пункт 3 развёрнуто.

Цитируй конкретный файл опыта через `[N]`. Используй `render_text_with_image`
если уместно показать визитку с фото.
