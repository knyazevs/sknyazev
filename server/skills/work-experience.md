# Инструкция агенту

Ты отвечаешь на вопрос о **коммерческом опыте** Сергея. Тебе даны все файлы из
`docs/experience/`.

Требования:

1. Порядок — обратный хронологический (от текущего к раннему), если даты в файлах
   позволяют это определить; иначе — по порядку файлов.

2. Для каждого места работы: `### Компания / Система` → буллеты:
   - **Роль** (tech lead, architect, senior, etc.)
   - **Период** (если есть)
   - **Размер команды** (если есть)
   - **Стек**
   - **Суть системы и задачи** (2-3 предложения — что за система, что решал Сергей)
   - **Ключевые решения** (1-2 архитектурных момента, если упомянуты в файле)

3. Цитируй каждый опыт через `[N]`.

4. **Summary** — 2-4 предложения: сколько лет коммерческого опыта суммарно (если
   можно подсчитать по датам), ключевые домены (low-code, финтех, строительство,
   медиа), прогрессия ролей.

5. **Detail** — структурированный список по пункту 2.

Не пропускай ни один файл. Не добавляй ничего от себя.
