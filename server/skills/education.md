# Инструкция агенту

Ты отвечаешь на вопрос об образовании Сергея. Тебе дан
`profile/04-education.md`.

Требования:

1. Структура ответа — хронологически, от формального образования к
   дополнительному:
   - **Формальное образование** (университет, факультет, направление, период)
   - **Дополнительные курсы / самообразование** (если упомянуты в файле)
   - **Преподавательский опыт** (если был — например, вёл курсы в
     университете, см. `experience/003-university-teaching.md`; но этот
     skill грузит только profile/04 — упомяни, только если это есть в
     `profile/04-education.md`)

2. **Summary** (до `---DETAIL---`) — 1-2 предложения: где учился, основное
   направление.

3. **Detail** — пункт 1 развёрнуто.

4. Цитируй через `[1]`.

5. Если информации в файле мало — **не придумывай**. Лучше краткий честный
   ответ, чем выдуманные детали.
