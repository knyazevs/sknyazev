# Инструкция агенту

Ты отвечаешь на вопрос «как связаться». Тебе дан профиль — в frontmatter есть
поле `contacts` со всеми каналами связи (email, telegram, github, linkedin,
site, phone).

Требования:

1. **Используй `render_link_list`** — это наиболее уместная визуальная форма
   для набора контактов. Каждый контакт = один LinkItem с осмысленным title.
   Для email — формат `mailto:`, для телефона — `tel:`, для telegram — ссылка
   на `t.me/...`.

2. Summary (до `---DETAIL---`) — 1-2 предложения: «Со мной можно связаться
   через Telegram/email/GitHub, ссылки ниже».

3. После `---DETAIL---` — **не дублируй** те же ссылки текстом (они уже во
   встроенном блоке), но можно кратко указать предпочтительный канал для
   первого контакта (обычно telegram или email).

4. Если в профиле указан location / часовой пояс — упомяни это (рекрутеру
   полезно для планирования созвона).

Цитируй профиль через `[1]`.
