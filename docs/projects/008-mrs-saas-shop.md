# МРС SaaS Shop — Магазин продуктов с автоматическим развёртыванием

## Контекст системы

Продукт внутри экосистемы МРС: магазин для покупки продуктов платформы
с автоматическим развёртыванием. SaaS в классическом смысле —
клиент покупает продукт и получает готовую среду.

## Роль

Участие в проектировании и разработке. Основной фокус — бэкенд.

## Ключевой эксперимент: Kotlin Native на бэкенде

В рамках R&D-направления бэкенд разрабатывался на **Kotlin Native** —
нетривиальный выбор для продакшен-системы.

**Что реализовал:**
- Интеграция **Apache Kafka** на Kotlin Native — нестандартная задача,
  т.к. большинство Kafka-клиентов ориентированы на JVM
- **Слой абстракции для работы с БД** на Kotlin Native —
  собственный ORM-подобный слой поверх нативных драйверов

Это один из немногих задокументированных случаев применения
Kotlin Native в серверном контексте с реальными нагрузками.

## Что это даёт

Отсутствие JVM-рантайма: меньший footprint, быстрый старт.
Цена — значительно меньшая зрелость экосистемы по сравнению с JVM.
