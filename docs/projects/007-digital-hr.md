# DigitalHR — Цифровой HR для команд

## Что это

Система для автоматизации HR-процессов внутри компании.
Снижает нагрузку на HR и упрощает жизнь сотрудников.

## Возможности

### Онбординг
Упрощённый онбординг новых сотрудников на базе базы знаний компании.
Новый человек получает нужную информацию автоматически, без HR как посредника.

### Пульс команды
Периодические опросы настроения и проблем — анонимный сбор обратной связи
для понимания состояния команды.

### Аналитика
Отслеживание показателей: вовлечённость, настроение, динамика по времени.

## Идея

HR как система, а не как человек.
Рутинные HR-процессы автоматизируются, освобождая HR для реальной работы с людьми.
