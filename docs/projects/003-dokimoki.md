# DokiMoki — Умное хранилище документов в Telegram

## Что это

Telegram-бот для хранения и управления личными документами:
чеки, гарантийные талоны, договоры, квитанции.

Загрузил — нашёл — не потерял.

## Возможности

### Хранение документов
- Загрузка изображений и файлов
- Автоматическое распознавание содержимого через LLM
- Поиск нужного чека или документа по смыслу

### Финансы и бюджет
- Отслеживание регулярных платежей
- Учёт расходов семьи
- Бюджетирование
- Отчётность (в т.ч. для бухгалтерии)

### Управление сроками
- Отслеживание дат и дедлайнов (гарантии, договоры)
- Уведомления об истекающих сроках

### Экспорт
- Доступ ко всем документам
- Экспорт в PDF

## Стек

| Слой | Технология |
|------|-----------|
| Bot framework | aiogram 3.x |
| Database | SQLite (dev) / PostgreSQL (prod) |
| Migrations | Alembic |
| FSM storage | MemoryStorage / Redis |
| File storage | Telegram file_id / S3 (MinIO) |
| Scheduler | APScheduler |
| OCR | Tesseract + Claude / GPT-4V / Google Vision |
| Payments | Telegram Stars |
| CI/CD | GitHub Actions + GHCR |

## Ключевая идея

LLM как слой понимания документов:
не просто хранилище файлов, а система, которая понимает что лежит внутри
и может отвечать на вопросы типа «найди чек на холодильник» или
«сколько потратили на продукты в марте».
