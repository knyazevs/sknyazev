# Инструменты и инфраструктура

## CI/CD

- **GitHub Actions** — пайплайны сборки, тестирования, деплоя
- **Jenkins** — опыт в production-проектах
- **GitHub Packages** — управление артефактами

## Инфраструктура

- **GitHub** — миграция команды, настройка инфраструктуры репозитория
- **Selectel** — облачный хостинг, передача резервного копирования провайдеру
- **VPS** — развёртывание backend-сервисов
