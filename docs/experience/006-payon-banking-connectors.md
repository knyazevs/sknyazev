# ACI Worldwide — PayON, платёжная платформа

## Контекст системы

Международный продукт для процессинга платежей.
Интеграции с банковскими системами через коннекторы.

Стек: Java.

## Роль

Бэкенд-разработчик.

## Ключевые задачи

- **Разработка коннекторов** к банковским системам — интеграция платёжной
  платформы с внешними банковскими API и протоколами
- **Внутренний инструмент для работы с логами** — разработка
  инструментария для анализа и диагностики логов платформы
- Поддержка и починка тестов
- Общая поддержка и развитие платформы

## Контекст

Первый серьёзный коммерческий опыт в enterprise Java-разработке.
Работа с реальными банковскими интеграциями и требованиями к надёжности
финансовых систем.
